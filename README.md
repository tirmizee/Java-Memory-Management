# Java-Memory-Management

## GC
 คือกระบวนการเรียกคืนหน่วยความจำที่ไม่ได้ใช้งานขณะ Runtime โดยอัตโนมัติ
 - จัดสรรหน่วยความจำ
 - ตรวจสอบว่าวัตถุที่อ้างอิงใด ๆ ยังคงอยู่ในหน่วยความจำ
